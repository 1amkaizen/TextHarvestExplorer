# TextHarvestExplorer

Tools untuk mining text yang setiap filenya berada di folder. 

Contoh strukturnya

 - Myolder
   - subfolder1
     - file1
     - file2
     - file3
   - subfolder2
     - file1
     - file2
     - file3
   - subfolder3
     - file1
     - file2
     - file3
    
### cara penggunaan
```
./TextHarvestExplorer Myfolder
```
